# XSY-University-trade
基于bmob   的校园     闲置物品交易   ，论坛 ，聊天，个人中心管理。线上交流，线下交易
 <br />

用到的技术：
 <br />
腾讯bugly集成跟踪crash
 <br />
butterknife注入
 <br />
CircleImageView圆形图像裁剪
 <br />
Rxvolley网络请求
 <br />
PictureSelector图片选择器
 <br />
环信easeUI聊天功能的集成
 <br />
当然最重要的是bmob,实现用户管理，论坛实现，文件上传与下载
